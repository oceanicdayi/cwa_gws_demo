# CWA GWS Demo - 氣象署 AI 自動化工作坊

## 🌐 GitHub Pages

此專案已設定 **GitHub Pages 自動部署**！

### 如何啟用：

1. **前往仓库设置**：
   - 点击仓库右上角 **Settings** → **Pages**

2. **选择部署来源**：
   - **Branch**: 选择 `main`
   - **Folder**: 选择 `/` (root)

3. **保存后**，您的網站將自動部署！

### 🔗 网站地址
部署完成後，網站將可以通過以下網址訪問：

```
https://oceanicdayi.github.io/cwa_gws_demo/
```
