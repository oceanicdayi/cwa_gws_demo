# CWA GWS Demo - 氣象署 AI 自動化工作坊

## 🌐 GitHub Pages

此專案已設定 **GitHub Pages 自動部署**！

### 如何啟用：

1. **前往倉庫設定**：
   - 點擊倉庫右上角 **Settings** → **Pages**

2. **選擇部署來源**：
   - **分支**：選擇 `main`
   - **資料夾**：選擇 `/` (根目錄)

3. **儲存後**，您的網站將自動部署！

### 🔗 網站網址
部署完成後，網站將可以透過以下網址存取：

```
https://oceanicdayi.github.io/cwa_gws_demo/
```
