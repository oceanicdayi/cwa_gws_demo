AI Agent 實作案例：Raspberry Pi 採購清單

場景：採購文件整理 | 工具：讀檔 · 計算 · Excel | 產出：採購清單.xlsx

案例背景

專案資料夾僅有一份參考清單 ref.txt。使用者請 AI Agent 閱讀內容、產出採購清單，並完成加總與預估後輸出成 Excel，適合作為「讀檔 → 推理 → 產出」的教學示範。

原始檔案內容

Raspberry Pi 5 Model B / 8GB 開發板 : 6125元
樹莓派 Pi 5 官方原廠 Type-C 27W 電源供應器 : 480元
樹莓派 Pi 5 被動式散熱外殼 (銀色): 600元

----------------------------------------------------------

Raspberry Pi 500 個人電腦套件(US): 7305元
樹莓派官方 15.6 吋螢幕 : 4000元

互動過程

User: 閱讀專案內容，提供採購清單

Agent: 彙整品名、單價、購買連結，標註方案組合，請使用者確認

User: 全部都買，幫我算總價之餘，再算一個採購預估金額(總價*1.2)

Agent: 依清單項目加總，套用 20% 緩衝係數產生採購預估金額

User: 輸出至 excel 檔

Agent: 產出：採購清單.xlsx（使用 Python openpyxl 產生試算表）

教學重點

這個案例短而完整，適合示範 AI Agent 如何把非結構化文字變成可交付成果：

  1. 讀取專案脈絡：先掃描目錄、讀檔，再回答——避免憑空假設
  2. 結構化輸出：把純文字清單轉成表格、方案對照
  3. 遵循明確規則計算：「全部都買」+「總價 × 1.2」是可驗證指令
  4. 落地交付物：不只聊天回覆，還產出 Excel，完成「問答 → 檔案」閉環
  5. 多輪迭代：清單 → 計算 → 匯出，每輪意圖清楚
  6. 可複現作業：輸入與輸出都在專案內，學員可自行重跑比對
