雲端生態與 Gemini

  • Gmail & Google Drive：基礎資料流、文件儲存與交換中心。
  • Gemini App 擴充功能：可直接檢視 Gmail 內容、生成 Google 文件、甚至設定排程定時工作。
  • Google Apps Script (GAS)：作為自動化的黏著劑，將 Sheet 的變更轉化為實際的檔案搬移與通知。

地端本機 AI (Antigravity)

  • Google Drive 電腦版：讓本機可以直接存取、監控雲端硬碟的檔案變更。
  • Antigravity CLI：強大的本機免費 AI 工具。可以直接叫 AI 整理資料夾、閱讀目錄內的所有檔案後生成說明。
  • RAG：結合專業資料庫，撰寫特定專案（如：立院回應稿），甚至撰寫網頁並部署。

實作實驗室

90 分鐘實作挑戰

Lab 1 設定 Gemini App 排程

拿起您的手機開啟 Gemini App，利用「排定的動作」建立自動化流程：

Prompt 提示詞範例：
「請定時（每小時一次）尋找我的未讀信件，將信件完整含附件寫入 Google Drive 中名為 gmail待處理區 的目錄內，並生成 Google Apps Script 程式碼，部屬在 Google Sheet 的擴充功能上，用於後續的自動化處理。」
  • 確保 Gemini 已經開啟 Workspace 擴充功能權限
  • 這會自動將您的信件完整儲存至指定的 Google Drive 目錄

Lab 2 在 Google Sheet 設定每小時排程

開啟網頁版 Google 試算表，點擊上方選單 擴充功能 > Apps Script，將以下程式碼貼上：

Code.gs 
function processGmailFiles() {
  // 1. 取得 gmail待處理區 目錄
  var folderName = "gmail待處理區";
  var folderIterator = DriveApp.getFoldersByName(folderName);
  var gmailFolder = folderIterator.hasNext() ? folderIterator.next() : DriveApp.createFolder(folderName);
  
  // 2. 取得目錄中的所有檔案
  var files = gmailFolder.getFiles();
  var processedCount = 0;
  
  while (files.hasNext()) {
    var file = files.next();
    var fileName = file.getName();
    var fileDate = file.getDateCreated();
    
    // 3. 這裡可以加入處理邏輯，例如：
    // - 記錄檔案資訊到 Sheet
    // - 發送通知
    // - 移動檔案到其他目錄
    
    // 4. 標記為已處理（可選：移動到歸檔目錄）
    var archiveFolderName = "gmail歸檔區";
    var archiveFolderIterator = DriveApp.getFoldersByName(archiveFolderName);
    var archiveFolder = archiveFolderIterator.hasNext() ? archiveFolderIterator.next() : DriveApp.createFolder(archiveFolderName);
    
    // 將檔案移動到歸檔目錄
    file.moveTo(archiveFolder);
    processedCount++;
  }
  
  // 5. 發送處理完成通知
  var emailAddress = Session.getActiveUser().getEmail();
  var emailSubject = "【系統通知】本次處理了 " + processedCount + " 個信件";
  var message = "系統已自動處理 gmail待處理區 目錄中的檔案，共處理 " + processedCount + " 個信件。

請前往本機的同步資料夾查看，並啟動 antigravity 進行後續處理。";
  
  MailApp.sendEmail(emailAddress, emailSubject, message);
}
關鍵步驟：設定時間驅動觸發條件

程式碼貼上儲存後，務必點擊左側「時鐘圖示」，新增觸發條件，選擇活動類型為 「時間驅動 (Time-driven)」，設定為每小時執行一次，並完成權限審查。

Lab 3 本機落地體驗：Antigravity

最後一哩路，回到我們的辦公室電腦前：

軟體安裝路徑

Google Drive for desktop (Windows)

官方下載路徑：

https://www.google.com/drive/download/

安裝後預設路徑：
C:\Users\[您的使用者名稱]\Google Drive\

Antigravity IDE

官方下載路徑：

https://antigravity.ai/download

Windows 安裝後路徑：
C:\Program Files\Antigravity\Antigravity.exe

Antigravity CLI

命令列工具安裝：

npm install -g @google/antigravity

全域安裝路徑：
C:\Users\[您的使用者名稱]\AppData\Roaming\npm\antigravity.cmd

驗證安裝：
antigravity --version

1. 本機同步

安裝 Google Drive 電腦版，確保 gmail待處理區 目錄已出現在您的電腦中（預設路徑：C:\Users\[您的使用者]\Google Drive\gmail待處理區）。

2. 啟動 Antigravity

開啟命令提示字元 (CMD) 或 PowerShell，導航至同步資料夾，使用 antigravity 命令監控 gmail待處理區 目錄，自動撰寫回覆信件、特定事件處理。

3. 歸檔處理

處理完的信件自動移至歸檔目錄，確保 gmail待處理區 目錄內信件清空，完成自動化閉環！
