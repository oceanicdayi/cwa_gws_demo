我們的 AI 武器庫：雲地結合

突破單一工具的限制，將雲端的廣泛連結與地端的深度運算完美結合。

雲端生態與 Gemini

  • Gmail & Google Drive：基礎資料流、文件儲存與交換中心。
  • Gemini App 擴充功能：可直接檢視 Gmail 內容、生成 Google 文件、甚至設定排程定時工作。
  • Google Apps Script (GAS)：作為自動化的黏著劑，將 Sheet 的變更轉化為實際的檔案搬移與通知。

地端本機 AI (Antigravity)

  • Google Drive 電腦版：讓本機可以直接存取、監控雲端硬碟的檔案變更。
  • Google Antigravity CLI：強大的本機免費 AI 工具。可以直接叫 AI 整理資料夾、閱讀目錄內的所有檔案後生成說明。
  • 領域知識技能 (RAG)：結合專業資料庫，撰寫特定專案（如：立院回應稿），甚至撰寫網頁並部署。

Lab 1 設定 Gemini App 排程

拿起您的手機開啟 Gemini App，利用「排定的動作」建立一個每日排程：

Prompt 提示詞範例：
「每天早上 9 點，請檢查我的未讀信件，將信件分類（例如：立法院業務、資安管理等）、摘要主旨、寄件者、日期，新增一行資料到我 Google Drive 裡名為『未讀信件列表』的 Google 試算表中。」
  • 確保 Gemini 已經開啟 Workspace 擴充功能權限。
  • 這會自動在指定時間，將您的信件轉換為結構化資料寫入 Sheet。

Lab 2 串接 Google Sheet 與 GAS 自動搬檔

開啟網頁版 Google 試算表，點擊上方選單 擴充功能 > Apps Script，將以下程式碼貼上：

Code.gs 
function onSheetChange(e) {
  // 1. 確保是新增資料引起的變動
  if (!e || (e.changeType !== 'EDIT' && e.changeType !== 'INSERT_ROW')) return;

  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var lastRow = sheet.getLastRow();
  
  // 避免重複執行，檢查 E 欄狀態
  var status = sheet.getRange(lastRow, 5).getValue();
  if (status === "已處理") return; 
  
  // 取得最新一筆的資料 (假設 A=類型, B=主旨, C=寄件者)
  var type = sheet.getRange(lastRow, 1).getValue();
  var subject = sheet.getRange(lastRow, 2).getValue();
  
  // 2. 執行檔案搬移 (DriveApp)
  var targetFolderName = "gmail工作區";
  var folderIterator = DriveApp.getFoldersByName(targetFolderName);
  var targetFolder = folderIterator.hasNext() ? folderIterator.next() : DriveApp.createFolder(targetFolderName);
  
  var rootFolder = DriveApp.getRootFolder();
  var safeSubject = subject.toString().replace(/'/g, ""); 
  var filesIterator = rootFolder.searchFiles("title contains '" + safeSubject + "'");
  
  var movedCount = 0;
  while (filesIterator.hasNext()) {
    var file = filesIterator.next();
    file.moveTo(targetFolder);
    movedCount++;
  }

  // 3. 發送 Email (MailApp)
  var emailAddress = Session.getActiveUser().getEmail(); // 寄給自己
  var emailSubject = "【系統通知】收到新信件並已搬移檔案：" + subject;
  var message = "系統已自動處理新信件：

" +
                "信件類型：" + type + "
" +
                "信件主旨：" + subject + "
" +
                "已搬移相關檔案數量：" + movedCount + " 個。

" +
                "請前往本機的同步資料夾查看，並啟動 antigravity cli 進行後續處理。";
                
  MailApp.sendEmail(emailAddress, emailSubject, message);
  
  // 4. 標記為已處理
  sheet.getRange(lastRow, 5).setValue("已處理");
}
關鍵步驟：設定安裝型觸發條件

程式碼貼上儲存後，務必點擊左側「時鐘圖示」，新增觸發條件，選擇活動類型為 「變更時 (On change)」，並完成權限審查。

Lab 3 本機落地體驗：Antigravity CLI

最後一哩路，回到我們的辦公室電腦前：

軟體安裝路徑

Google Drive for desktop (Windows)

官方下載路徑：

https://www.google.com/drive/download/

安裝後預設路徑：
C:\Users\[您的使用者名稱]\Google Drive\

Antigravity IDE

官方下載路徑：

https://antigravity.ai/download

Windows 安裝後路徑：
C:\Program Files\Antigravity\Antigravity.exe

Antigravity CLI

命令列工具安裝：

npm install -g @google/antigravity

全域安裝路徑：
C:\Users\[您的使用者名稱]\AppData\Roaming\npm\antigravity.cmd

驗證安裝：
antigravity --version

1. 本機同步

安裝 Google Drive 電腦版，確保「gmail工作區」資料夾已出現在您的電腦中（預設路徑：C:\Users\[您的使用者]\Google Drive\gmail工作區）。

2. 啟動 Antigravity

開啟命令提示字元 (CMD) 或 PowerShell，導航至同步資料夾，使用 antigravity 命令監控該資料夾。

3. 人工放行

檢核 AI 產出的「回應稿.pdf」，將其拖曳至「已核准」資料夾，自動完成回信！
